# Contributing to terminal-ime-proxy

Thank you for your interest in contributing!

## How to Contribute

### Reporting Bugs

1. Check if the bug already exists in [Issues](https://github.com/d-init-d/terminal-ime-proxy/issues)
2. Create a new issue with:
   - Your OS and terminal
   - IME software (Unikey, Sogou, etc.)
   - Steps to reproduce
   - Expected vs actual behavior

### Testing

We need help testing with different:
- Languages (Vietnamese, Chinese, Japanese, Korean, Thai, etc.)
- IME software
- Terminal emulators
- Operating systems

### Code Contributions

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/my-feature`
3. Make your changes
4. Run tests: `npm test`
5. Commit: `git commit -m "feat: add my feature"`
6. Push: `git push origin feature/my-feature`
7. Create a Pull Request

### Code Style

- Use TypeScript
- Follow existing code patterns
- Add tests for new features
- Update documentation as needed
